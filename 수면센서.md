# 🎙️ 오디오 기반 수면 단계 추정을 위한 Reference 센서 분석

오디오 녹음 시 Ground Truth(정답 레이블)로 활용할 수 있는 센서들을 **정확도 계층**에 따라 분석합니다.

---

## 🥇 Tier 1 — 절대 골드 스탠다드: 임상 PSG (Polysomnography)

PSG는 EEG, EOG(안구전도), EMG(근전도), 호흡 노력, 기류, 맥박, 혈중 산소포화도 등 다중 신호를 포착하는 수면 평가의 절대 임상 표준이며, 전문 분류자 간 평균 일치율은 약 82.6%에 달합니다.

| 항목 | 내용 |
|------|------|
| **측정 신호** | EEG (6채널), EOG, 턱·다리 EMG, ECG, SpO₂, 호흡벨트, 기류 |
| **수면 단계 분류** | Wake / N1 / N2 / N3(Deep) / REM (5단계) |
| **정확도** | 인간 전문가 간 일치율 82~89% |
| **표준 프레임워크** | AASM 기준, 30초 epoch 단위 스코어링 |
| **단점** | 수면 클리닉 필요, 전극 부착 복잡, 자연 수면 환경 구현 어려움 |

> ✅ **오디오 연구에서의 활용**: 오디오 + PSG를 병원 수면 클리닉에서 동시 녹음하면 가장 신뢰도 높은 레이블 확보 가능. 단, 비용/피험자 부담 높음.

---

## 🥈 Tier 2 — 휴대용 EEG 웨어러블 (최선의 가정용 대안)

PSG의 현실적 대안으로, **뇌파를 직접 측정**하여 다른 센서류보다 본질적으로 정확합니다.

### ① **Dreem 3S (Beacon Biosignals)** ⭐ 가장 추천

Dreem 헤드밴드의 자동 수면 단계 알고리즘은 전체 정확도 85.76%로 인간 수준의 성능에 근접했으며, 단계별로는 N1: 56%, N2: 88%, N3: 85%, REM: 92%, Wake: 85%를 달성했습니다.

6개 웨어러블 비교 연구에서 Dreem은 2단계 kappa = 0.76, 4단계 kappa = 0.76~0.86으로 가장 높은 성능을 보였으며, 최고 정확도 수면 단계 분류가 필요한 경우 EEG 기반 Dreem이 권장됩니다.

Dreem 3S는 2023년 8월 FDA 510(k) 승인을 받았으며, 6개의 건식 EEG 전극과 가속도계를 탑재해 환자 가정에서 임상급 EEG 데이터를 수집합니다. 2024년 12월에는 FDA로부터 알고리즘 지속 개선을 위한 PCCP 승인도 받았습니다.

| 항목 | 내용 |
|------|------|
| **측정 신호** | 건식 EEG 6채널 (O1, O2, FpZ, F7, F8) + 가속도계 + PPG |
| **수면 단계** | Wake / N1 / N2 / N3 / REM (5단계) |
| **4단계 Kappa** | 0.76~0.86 |
| **규제 승인** | FDA 510(k) 승인 (2023), 유럽 CE |
| **현재 판매** | B2B/연구기관 전용 (Beacon Biosignals 통해 접근) |
| **오디오 연구 적합성** | ✅ 매우 높음 - 가정 환경 가능, EEG 직접 측정 |

---

### ② **Somfit**

Somfit는 43개 검증 연구를 분석한 메타분석에서 가장 많이 연구된 상업용 wEEG 기기 중 하나로, PSG와 중등도~높은 일치율을 보였습니다. 오스트레일리아 기반의 연구/임상용 EEG 웨어러블로, raw EEG 데이터 접근이 가능합니다.

---

### ③ **Muse S Gen 2 (현재 유일한 소비자용 EEG 헤드밴드)**

Dreem이 2021년 소비자 판매를 중단하고 의료/연구 부문으로 전환함에 따라, Muse S Gen 2는 현재 EEG 기반 수면 트래킹이 가능한 사실상 유일한 소비자 옵션입니다. 단, 검증 연구가 Dreem 대비 부족하며 PSG 대비 정확도는 다소 낮습니다.

---

## 🥉 Tier 3 — PPG + 가속도계 소비자 웨어러블

뇌파를 직접 측정하지 않고 심박수/HRV/움직임을 통해 수면 단계를 **간접 추정**하는 방식입니다.

### ① **Oura Ring Gen 3/4** — 소비자용 중 최고 정확도

Brigham and Women's Hospital 연구에서 Oura Ring은 4단계 수면 분류에서 가장 정확한 소비자용 수면 트래커로 평가받았습니다. Wake 감도 68.6%(Fitbit 67.7%, Apple Watch 52.4%)와 Deep sleep 감도 79.5%(Fitbit 61.7%, Apple Watch 50.5%)로 비교 기기 중 가장 높았습니다.

| 항목 | 내용 |
|------|------|
| **측정 신호** | PPG (HR, HRV, SpO₂), 가속도계, 피부온도 |
| **수면 단계** | Wake / Light (N1+N2) / Deep (N3) / REM |
| **강점** | 4단계 과추정/과소추정 없음, 체온·일주기리듬 통합 |
| **오디오 연구 적합성** | ✅ 높음 - 착용 편리, 수면 방해 적음, 가정용 |

---

### ② **Fitbit Sense 2 / Charge 5**

Oura Ring 3와 Fitbit Sense 2는 이전 세대 대비 수면 단계 감지 정확도가 향상되었으며, 특히 심층 수면(Deep Sleep) 분류에서 웨어러블이 효과적임을 보여줍니다.

---

### ③ **기타 (참고용)**

| 기기 | 특이사항 |
|------|---------|
| Apple Watch Series 9 | 라이트 슬립 과추정 (~45분), Deep 과추정 (~43분) — 정확도 낮음 |
| Whoop 4.0 | HRV 측정 우수, 수면 단계 정확도 중간 |
| Garmin Vivosmart 4 | Wake, Deep 오차 크고 연구 사용에 부적합 |

## 🎯 오디오 연구 목적별 센서 추천

| 목적 | 추천 센서 | 이유 |
|------|-----------|------|
| **학술 논문 / 최고 신뢰도 필요** | PSG + 수면 클리닉 동시 녹음 | 최고 정밀도의 레이블 확보 |
| **가정 환경 / 대규모 피험자** | Dreem 3S | EEG 직접 측정, FDA 승인, 가정 사용 가능 |
| **예산 제한 / 쉬운 피험자 모집** | Oura Ring Gen 3/4 | 착용 편리, 가격 합리적, 소비자용 중 최고 정확도 |
| **연구실 + 편의성 균형** | Dreem 3S + 동시 오디오 녹음 | 임상급 EEG + 자연 수면 환경 동시 구현 |

## 💡 핵심 결론

오디오 기반 수면 단계 추정 연구를 위한 **최선의 레퍼런스 센서는 Dreem 3S**입니다. PSG의 85.76% 수준의 정확도를 가정 환경에서 구현하면서, EEG를 직접 측정하기 때문에 PPG 기반 기기 대비 본질적으로 신뢰도 높은 레이블을 제공합니다. 다만 연구기관용으로만 판매되므로, 대규모 소비자 피험자 연구에는 **Oura Ring Gen 3/4**가 현실적인 최선의 대안입니다. 최고의 레이블 품질이 필요하다면 수면 클리닉에서 **PSG + 마이크 동시 녹음**을 진행하는 방법도 고려해볼 수 있습니다.
